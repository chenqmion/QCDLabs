# QCDLabs
This package is designed for plug-and-play use of instruments for experiments in QCD Labs. It's shallow, flexible, easy to use and hard to make mistakes.

## Instrument [class]
Instrument drivers based on the socket method (some on pyvisa).

## Pattern [class]
Combinations of instruments to realize selected task.

## Test
Temporary files for debugging. They can also be used as example scripts.

## Lab
Real measurement script and data.
